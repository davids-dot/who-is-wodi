## ADDED Requirements

### Requirement: 描述全局唯一性
系统 SHALL 确保任何玩家在任何轮次生成的描述，不能与之前任何轮次任何玩家（包括该玩家自己）生成过的任何描述重复。

#### Scenario: 同一玩家跨轮次不重复
- **GIVEN** 第1轮老王生成了描述 "夏日的噪音"
- **WHEN** 第2轮轮到老王描述
- **THEN** 老王生成的描述不能是 "夏日的噪音"

#### Scenario: 不同玩家之间不重复
- **GIVEN** 第1轮老王生成了描述 "夏日的噪音"
- **WHEN** 第2轮小美描述
- **THEN** 小美生成的描述不能是 "夏日的噪音"

#### Scenario: 当前轮次内不重复
- **GIVEN** 第2轮前面已有玩家生成了描述 "树上叫个不停"
- **WHEN** 当前玩家生成描述
- **THEN** 该描述不能是 "树上叫个不停"

#### Scenario: Prompt 明确列出历史描述
- **WHEN** 构建描述生成 prompt
- **THEN** prompt 中 SHALL 包含所有历史轮次的所有描述记录
- **AND** prompt 中 SHALL 明确说明"不能重复以下任何描述，包括你自己之前说过的"
