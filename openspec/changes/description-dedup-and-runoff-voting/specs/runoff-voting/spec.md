## ADDED Requirements

### Requirement: 决胜投票触发条件
当投票统计出现多个最高票候选人时，系统 SHALL 触发决胜投票，由非最高票玩家对最高票候选人进行第二轮投票。

#### Scenario: 单一最高票直接淘汰
- **GIVEN** 投票结果：老王3票，小美2票，阿强1票
- **WHEN** 系统统计票数
- **THEN** 老王被淘汰（最高票且唯一）
- **AND** 不触发决胜投票

#### Scenario: 多最高票触发决胜投票
- **GIVEN** 投票结果：老王2票，小美2票，阿强1票，大刘1票，Lily1票，老张1票
- **WHEN** 系统统计票数
- **THEN** 识别最高票集合 M = {老王, 小美}
- **AND** 识别非最高票集合 N = {阿强, 大刘, Lily, 老张}
- **AND** 触发决胜投票，让 N 中的玩家对 M 中的候选人投票

#### Scenario: 全部平票无法决胜
- **GIVEN** 投票结果：老王1票，小美1票，阿强1票，大刘1票，Lily1票，老张1票
- **WHEN** 系统统计票数
- **THEN** 识别最高票集合 M = {所有人}
- **AND** 非最高票集合 N 为空
- **AND** 判定为平票，无人淘汰，进入下一轮

### Requirement: 决胜投票执行流程
决胜投票 SHALL 复用现有投票机制，但候选人限制为最高票集合中的玩家。

#### Scenario: 决胜投票执行
- **GIVEN** 最高票集合 M = {老王, 小美}
- **AND** 非最高票集合 N = {阿强, 大刘, Lily, 老张}
- **WHEN** 执行决胜投票
- **THEN** 只有 N 中的玩家参与投票
- **AND** 投票候选人只能是 M 中的玩家
- **AND** 投票完成后重新统计票数

#### Scenario: 决胜投票后仍有结果
- **GIVEN** 决胜投票结果：老王3票，小美1票
- **WHEN** 系统统计决胜投票票数
- **THEN** 老王被淘汰（决胜投票产生唯一最高票）

#### Scenario: 决胜投票后仍平票
- **GIVEN** 决胜投票结果：老王2票，小美2票
- **AND** 已无其他玩家可参与决胜
- **WHEN** 系统统计决胜投票票数
- **THEN** 判定为平票，无人淘汰，进入下一轮
